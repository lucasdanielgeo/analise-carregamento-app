# Analise Carregamento App
Este painél Painel de informações sobre a pesquisa de do carregamento das linhas de onibus de Florianópolis duranto os dias úteis.

Pacotes python utilizados nesse web app:
 - streamlit
 - pandas
 - numpy
 - plotly
